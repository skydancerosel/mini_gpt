# Reproducing: Spectral Edge Dynamics of Training Trajectories

**arXiv:** [2603.15678](https://arxiv.org/abs/2603.15678)

## Overview

This paper introduces Spectral Edge Dynamics (SED): rolling-window SVD of parameter updates reveals a sharp boundary between signal and noise. We test 6 properties across two model scales.

---

## Experiment 1: TinyStories 51M (Section 3)

### Setup

Train 4 seeds (10K steps each):
```bash
for seed in 42 123 149 256; do
    python training/pilot.py --seed $seed --wd 0.5 --lr 0.001 --steps 10000
done
```

## Experiment 2: GPT-2 124M (Section 4)

### Setup

This experiment uses GPT-2 124M with a FineWeb → OpenWebText distribution shift. You need:
1. GPT-2 124M checkpoints from a pretrain (FineWeb 10B, steps 0–17,600) then fine-tune (OpenWebText, steps 17,800–25,600)
2. Checkpoints every 200 steps (60 checkpoints from steps 13,800–25,600)
